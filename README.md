# 🤖 Agentic DSA Assistant

An intelligent AI assistant that helps with Data Structures and Algorithms using an agentic architecture.

## 🚀 Features
- Hint generation
- Concept explanation
- Code analysis
- Multi-intent handling
- Conversation memory
- Weak topic tracking

## ⚙️ Tech Stack
- Python
- Groq API (LLaMA 3.3)
- Streamlit

## ▶️ How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
